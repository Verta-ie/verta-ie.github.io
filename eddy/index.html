Eddy makes it easier to edit M3U lists for IPTV Services

Eddy is an opensource project, I hope that other people keep it flourishing and add to it.
Contribute

  • Covers the basics

    You get basic funcionality such as opening from file , url and saving the playlists, you can add a new channel or delete one, you also get the ability to test the channel to see if the stream works. You can double click the video to go fullscreen. Eddy is unfortunately windows only right now.

Eddy needs you

Anyone can add new features, I don't actively develop this software as I am too busy, but perhaps you can help!
